# Country Flag — Chrome Extension

Расширение показывает **флаг страны**, в которой вы находитесь, прямо как иконку в панели браузера. Местоположение определяется по вашему IP-адресу и обновляется **каждую минуту**.

---

## Возможности

- Иконка = флаг вашей страны (обновляется автоматически раз в минуту)
- Всплывающее окно — IP, страна, город, регион, время последнего обновления
- Кнопка ручного обновления
- Два независимых геолокационных API (один — резервный)
- Работает с VPN (показывает страну выходного узла)

---

## Установка

### Шаг 1 — Скопировать расширение на локальный компьютер

Откройте терминал **на вашем компьютере** и выполните:

```bash
scp -r root@185.181.8.39:/opt/chrome-flag-extension ~/Downloads/chrome-flag-extension
```

> Если вы уже на локальном компьютере с доступом к папке — пропустите этот шаг.

---

### Шаг 2 — Сгенерировать иконки по умолчанию

```bash
cd ~/Downloads/chrome-flag-extension
python3 create_icons.py
```

Скрипт создаст заглушку-глобус в папке `icons/`. После первого запуска расширение **сразу заменит** её на флаг вашей страны.

---

### Шаг 3 — Загрузить расширение в Chrome

1. Откройте Chrome и перейдите по адресу:
   `chrome://extensions/`

2. Включите **Режим разработчика** (переключатель в правом верхнем углу)

3. Нажмите **«Загрузить распакованное»**

4. Укажите папку `chrome-flag-extension`

5. Расширение появится в списке и сразу начнёт работу

---

### Шаг 4 — Закрепить на панели

1. Нажмите иконку пазла 🧩 справа от адресной строки
2. Рядом с **Country Flag** нажмите 📌
3. Иконка с флагом появится на панели инструментов

## Используемые сервисы

| Сервис | Назначение | Лимит |
|---|---|---|
| [ipwho.is](https://ipwho.is) | Определение IP → страна | Бесплатный, без лимитов |
| [ipapi.co](https://ipapi.co) | Резервный геолокационный API | 1 000 запросов/день |
| [flagcdn.com](https://flagcdn.com) | PNG-изображения флагов | Бесплатный |

## Обновление расширения

Если вы изменили файлы, нужно перезагрузить расширение:

1. Откройте `chrome://extensions/`
2. Найдите **Country Flag**
3. Нажмите кнопку 🔄 (обновить)

---

## Устранение неполадок

**Иконка не обновилась после установки**
→ Нажмите кнопку «Обновить сейчас» в popup-окне или перезагрузите расширение через `chrome://extensions/`

**Показывает «не удалось определить местоположение»**
→ Проверьте интернет-соединение. Если используете корпоративный прокси — оба API могут быть заблокированы.

**Неправильная страна**
→ Ожидаемо при использовании VPN — расширение показывает страну вашего IP-адреса (выходного узла VPN).

**Ошибка при загрузке (`manifest.json` не найден)**
→ Убедитесь, что вы выбираете именно папку `chrome-flag-extension`, а не вложенную папку.

## Разрешения

Расширение запрашивает только минимально необходимые разрешения:

| Разрешение | Зачем |
|---|---|
| `alarms` | Запуск обновления каждую минуту |
| `storage` | Кэш данных о местоположении между открытиями popup |
| `action` | Изменение иконки расширения |
| `https://ipwho.is/*` | Геолокация по IP |
| `https://ipapi.co/*` | Резервная геолокация |
| `https://flagcdn.com/*` | Загрузка изображений флагов |
